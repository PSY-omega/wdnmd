流行病学重点

流行病学：原意指研究人群中疾病流行的学科。

兽医流行病学：兽医流行病学主要研究动物群体中的疾病问题，是研究动物群体中疾病分布及其决定因素的一门学科。

流行病学四大研究方法：

  1. 描述流行病学
  2. 分析流行病学
  3. 实验流行病学
  4. 理论流行病学

地方流行：有两个方面的含义，其一是说明某地区动物群体中的某病以通常的、相对稳定的频率发；其二是表示该地区动物群体中该病的发生在畜群间、时间和空间分布上有一定的规律性。

大流行：指散布范围广，群体中受害动物比例大的流行。

病死率：是度量动物群体中较短时期内新发病例频数的指标，实际上是流行或暴发期间的发病比例。如饲料或饮水等共同来源造成的暴发或流行常使用罹患率。

死亡率：是描述一定观察时期内死亡动物总数于该群体同期动物平均数之比。

罹患率：为一定时期内患某病的动物因该病而死亡的频率。

三间分布：即疾病分布，是疾病频率分布的简称，是指疾病在畜群间、时间、空间的分布状况，又称三间分布。

生态系：由生物群落及其占据的自然环境所构成的相互依赖而又相互制约的复合体称为生态系。

小生境：在生态学极期中，生物群落的每一种生物对其竞争者、敌人或食物而言都占据着一适当而明确的位置，即小生境。小生境是一种生物在生物群落中所处的功能位置，即它所起的作用。

病因因素：又称疾病的决定因素，是指影响动物群体健康的任何因素，它与某些具体疾病有因果联系。

病因因素的分类：

  1. 按在致病中所起的作用分为主要因素和次要因素。
  2. 按因素是内在还是外在分为内在因素和外在因素。
  3. 按因素与宿主、致病因子和环境的联系分为宿主因素、致病因子因素和环境因素。

疾病的环境决定因素：包括地理位置、气候、植被、地质水文、饲养管理等。

（简答）疾病的宿主决定因素：

  1. 传染性宿主：传染源。病畜，病原携带者，受感染的其他种类的动物。
  2. 易感宿主：即缺乏免疫力的动物。
  3. 免疫宿主：即具有特异性免疫力和非特异性抵抗力的宿主。尽管传染性病原体在传染病的发生和发展中起重要作用，但宿主对入侵病原体的反应性是决定疾病能否发生的根本原因。

病因推断：病因推断涉及的是一种特殊的因果关系，即引起疾病的决定因素和发生疾病之间的关系。

病因推断的两个假设及其区别：

形成病因假设的方法：

  1. 求同法：如果某因素对存在某疾病的很多不同情况是公共的，该因素就可能是该病的病因。
  2. 求异法：如果两种情况下疾病的频率不同，在一种情况下存在该因素，而在另一种情况下缺乏该因素，则该因素可以被怀疑是因果性的。
  3. 伴随变异法：当某因素的频率和强度发生连续变化时，不同情况下的疾病频率也伴随发生变化，则该因素是疾病的假设因素。
  4. 类推法：当一种疾病的分布于另一种病因已研究清楚的疾病的分布相似时，则这两种疾病可能有相似的病因。

兽医流行病学资料的性质：

兽医流行病学资料的评价：

  1. 真实性：符合并反应实际情况。
  2. 可靠性：相同条件下，反复调查，获得同样结果的稳定程度。需统一调查研究结果，统一实验方法，统一培训调查人员。
  3. 完整性：是最终得到正确分析结论的基本保证。
  4. 可比性：用于比较的动物群体应具有基本一致的条件，在比较项目以外的各方面均有可比性。

哨兵监测：监测可能涉及全国范围内的所有畜群，也可能只针对少数几个牧场、屠宰场、兽医院或实验室。由于这些单位被用来观察某一疾病的发生，因此称为哨兵单位。(针对若干的代表单位来观察某一疾病的发生)

靶向监测：是指就某种疾病收集特定的信息，以便测量其在特定群体中的发生水平并监管其无病状态。

动物疫病监测的类型：疾病监测和流行病学监测、哨兵监测、血清学监测、被动监测和主动监测、靶向监测和扫描监测。

动物疫病的监测机制：自愿报告、强制报告、暴发调查、哨兵监测、结构式调查、普查。

疾病经济损失的评估：在动物生产单位，疾病所导致的经济损失通常包括以下方面:

死亡动物、药物费用、兽医费用、饲料转换率降低、生产水平下降。

兽医在动物卫生保健体系中的主要任务：

  1. 诊断和预防动物群体中的流行性疾病。
  2. 对环境因素如营养、畜舍和管理等与疾病和生产性能间的关系进行指导。
  3. 对畜禽生产技术和相关政策提出指导意见。

疾病的消灭、扑灭：

  • 疾病的消灭或根除指的是通过实施防控规划，使得疫病从流行或地方流行状态转变为完全无病状态。
  • 扑灭是指疫情发生早期，还处于小范围流行的情况下，采取各种措施使疫情终止。

暴发调查的步骤包括哪些：

  1. 速赴现场
  2. 提出假设
  3. 进一步调查研究与核实
  4. 检验结论

动物疫病的健康风险评估4个步骤

  1. 动物疾病的健康风险危害辨识
  2. 致病因子剂量—应答评估
  3. 致病因子暴露评估
  4. 风险特性确定

动物疫病的健康风险分析4个组成部分

  1. 风险识别
  2. 风险评估
  3. 风险控制与管理
  4. 风险信息交流

普查：是对群体中的所有成员进行的调查，又称全面调查。

结构式调查：

描述性研究（流行病学调查）的设计步骤：

  1. 明确调查目的
  2. 确定调查方法和内容
  3. 确定调查范围和方式
  4. 确定调查样本大小
  5. 计算置信区间
  6. 误差的控制
  7. 数据处理和总结

现况研究：是研究特定时点或时期和特定动物群体中的有关变量与疾病或健康状况的关系，即是调查这个特定群体中的个体是否患病和是否具有某些变量或特征的情况，从而探索具有不同特征的暴露情况与疾病或健康状况的关系。

相对危险性：暴露组的发病（死亡）率与非暴露组的发病（死亡）率之比值，称为相对危险性。

临床试验的分期：根据评价的目的和研究深入的程度按时间顺序分为四期

  1. 一期临床试验：在人体作药理学和安全性评价试验
  2. 二期临床试验：初步评价药物的有效性，进一步评价安全性
  3. 三期临床试验：进一步确定有效性，监测副作用
  4. 四期临床试验：检查在广泛使用条件下药物在不同人群的效果和远期或罕见的副作用，改进给药剂量等

临床试验的要素（可能大题）：处理因素、受试动物、试验效应

  1. 处理因素：在试验设计中，要研究的预防或治疗手段，要求在整个实验过程中应始终如一
  2. 受试动物：即靶动物，是指特定的经过种类、级别和品系鉴定并将给予被试药物的动物
  3. 试验效应：施加处理因素后，受试动物产生的效果

流行病学建模：是以数学模型或数学语言，表达疾病在群体流行过程中，各因素内在的和数量的关系，并对疾病事件进行预测

流行病学建模的方法：确定性模型（确定性微分模型、确定性矩阵模型）随机性模型（随机性微分模型、链形二项式模型）

流行病学模型的主要功能（简答）：

  1. 分析流行过程、预测流行趋势
  2. 定量研究各种流行因素的效应
  3. 设计和评价主动防控疾病的方案
  4. 检验病因假设
  5. 应用于教学和培训

试验的真实性、可靠性：试验的真实性或有效性，指筛检或诊断试验的测量值与实际情况符合的程度，即将有病动物与正常动物正确区分开来的能力。试验的可靠性指的是在重复同一试验时产生相似的结果，可靠性的特征是可重复性。

平行试验、系列试验的内涵（理解）：平行试验即同时采用两种以上的试验对动物进行检查，如果其中任何一种试验是阳性，则判断动物为感染。；系列试验中，前一个试验阳性的动物才进行下一个试验，所有试验均为阳性的动物才判定为感染。

分子流行病学：是将分子生物学的理论和技术应用于流行病学调查研究，在分子水平上阐明疾病或医学事件在畜群或环境生物群体中频率分布及其决定因素的学科。

分子流行病学的研究内容：

  1. 传染性疾病：病原体的分型和分类、传染源和传播途径分析、用于病原体起源和进化研究、用于细菌耐药性研究
  2. 肿瘤和心血管疾病
  3. 遗传和代谢疾病

比较流行病学：是指对一种动物疾病的调查研究可以提供对另一种动物或人类疾病的病因和发病机理有价值的信息。

比较流行病学4个生物模型：实验模型、阴性模型、孤儿模型、自发模型

多因素分析：主要用于从多种因素中确定疾病频率分布的决定因素，阐明复杂疾病过程中各变量间的关系和预测疾病发生水平或其他因素的水平。
