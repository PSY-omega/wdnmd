兽医寄生虫学串讲

总论

兽医寄生虫学：是研究寄生于家畜，家禽，伴侣动物和其他动物的各种寄生虫及其所引起疾病的科学。（根据寄生虫的分类可以分为兽医蠕虫学、兽医昆虫学、兽医原虫学）

生物间相对关系

  • 共栖、互利共生、偏利共生

  • 寄生：是两个生物体间复杂的相互关系。一种生物暂时或永久地寄居于另一种生物的体内或体表，以被寄生者的组织、体液等作为营养而获益，同时给被寄生者造成不同程度的损害，甚至造成死亡，营寄生生活的双方中必有一方得益，另一方受害，这种现象叫寄生。

    在这个关系中，寄生者称作寄生虫（parasite）被寄生者称为宿主（host）。

寄生虫的分类
  1. 内寄生虫与外寄生虫：从寄生部位来分。暂时地或永久地寄生在宿主体表的寄生虫为外寄生虫，在宿主体内的组织、器官、血液等处寄生的寄生虫为内寄生虫。
  2. 土源性寄生虫与生物源性寄生虫：从寄生虫的发育过程来分。寄生虫在其发育过程中不需要中间宿主参与，就可以完成整个发育和传播的过程。这类寄生虫的幼虫是在土壤中发育至感染期的，故称为土源性寄生虫。也称单宿主寄生虫。如蛔虫，球虫；寄生虫的整
    个发育过程需要有中间宿主的参与才能完成。也就是说其发育过程需要2个以上的宿主。其幼虫需通过另一种生物作为中间宿主进行发育，才能感染终末宿主，故称为生物源性寄生虫。也称多宿主寄生虫。如绦虫、吸虫。
  3. 长久性寄生虫与暂时性寄生虫：按寄生时间来分。长久性寄生虫，指终生或某一个生活阶段不能离开宿主，否则难以存活的寄生虫。如蛔虫 ; 暂时性寄生虫，指寄生虫只在采食时才暂时地寄生于宿主体表。如蚊 子、臭虫等。
  4. 专一宿主寄生虫与非专一宿主寄生虫：从寄生虫寄生的宿主范围来分。 专一宿主寄生虫：指有些寄生虫只寄生于某一特定的宿主，对宿主有严格的选择性。如球虫，马尖尾线虫。 非专一性宿主寄生虫：能够在2种以上的多种宿主寄生的寄生虫。如肝片形吸虫。
  5. 专性寄生虫与兼性寄生虫：从寄生虫对于寄生生活的适应程度来分。 专性寄生虫也称固需寄生虫，指寄生虫的寄生关系在其生活史中不可缺少，否则生活史就不能完成，生存完全依赖于寄生生活。如绦虫、吸虫。; 兼性寄生虫指既可自由生活又可寄生生活的寄生虫。如纳格里虫和类圆线虫。
宿主类型

  1. 终末宿主：蠕虫寄生虫的成虫或原虫有性繁殖阶段寄生的宿主称终末宿主。如人是猪带绦虫的终末宿主。

  2. 中间宿主：蠕虫寄生虫幼虫或原虫无性生殖阶段所寄生的宿主。如猪是猪带绦虫的中间宿主。

  3. 补充宿主：某些种类的寄生虫在发育过程中需要两个中间宿主，后一个中间宿主称为补充宿主。如双腔吸虫的补充宿主是蚂蚁。

  4. 贮藏宿主：也叫转运宿主或传递宿主。即宿主体内有寄生虫虫卵或幼虫存在，虽不发育繁殖，但保持着对易感动物的感染力，此动物被称为贮藏宿主。如蚯蚓是鸡异刺线虫的贮藏宿主。

  5. 保虫宿主：一些惯常寄生于某种宿主的寄生虫，有时也可寄生于其他一些宿主，但寄生不普遍、无明显危害，通常称这种不惯常被寄生的宿主为保虫宿主，其在流行病学上起一定作用。如耕牛是日本血吸虫的保虫宿主。野兔就是肝片吸虫的保虫宿主。

  6. 带虫宿主：有的寄生虫感染宿主后随着宿主抗体抵抗力的增强或通过药物治疗，使宿主处于隐性感染状态，但体内保留有一定数量的虫体，这样的宿主称为带虫宿主。宿主的这种状态称为带虫现象。它在临床上不表现症状，但宿主对同种寄生虫再感染具有一定的免疫力 一旦宿主体内的虫体完全消失，这种免疫力也随之消失。此称为带虫免疫 。如成年鸡感染球虫或往往不发病，但成为带虫者。

  7. 超寄生宿主：许多寄生虫是其他寄生虫的宿主，此种情况称为超寄生。如疟原虫在蚊子体内。

  8. 传播媒介：通常指在脊椎动物宿主间传播寄生虫病的一类动物，尤指传播血液寄生虫病的吸血节肢动物。如蚊子在人之间传播疟原虫，蜱在牛之间传播梨形虫等

寄生虫生活史

  • 概念：寄生虫的生长、发育和繁殖的全一个完整循环过程。也叫发育史。它包括了寄生虫的感染和传播。可分为两种类型：一种是不需中间宿主的生活史，又称直接发育型。一种是需要中间宿主的生活史，又称间接发育型。

  • 生活史完成的必要条件：①适宜的宿主、②发育至感染性阶段（或侵袭性阶段）③接触 ④感染途径 ⑤移行 ⑥寄生虫必须战胜宿主的抵抗力。

  • 宿主对寄生虫生活史的影响：

    ①年龄因素的影响：不同年龄的个体对寄生虫的易感性有差异，一般而言幼龄动物更易感。

    ②宿主体质及饲养管理情况的影响：宿主体质健壮，营养合理，饲养管理条件优良，对寄生虫的抵抗力就强，抗病能力就高。

    ③宿主免疫作用的影响：主要表现在两个方面：一是在寄生虫侵入、移行、寄生部位

寄生虫病的流行病学

  • 寄生虫病流行病学：研究动物群体中寄生虫病的发病原因和条件、传播途径、发生发展规律、流行过程及其转归等方面的特征的科学。

  • 流行的三要素：传染源、传播途径和易感动物。

    • 传染源（感染来源）通常是指寄生有某种寄生虫的[1~6]宿主

    • 传播途径（感染途径）是指病原从感染来源感染给易感动物所需要的方式。随寄生虫的种类不同而异，主要有以下几种：

      经口感染（多数寄生虫属此方式）、经皮肤感染（钩虫、血吸虫）、接触感染（外寄生虫，如蜱螨虱）、经节肢动物感染（一些血液原虫和丝虫）、经胎盘感染（弓形虫等）、自身感染（如猪带绦虫）。

  • 寄生虫生活史中能使动物机体感染的阶段称为感染性阶段；寄生虫侵入动物机体并能生活或长或短一段时间，这种现象称为寄生虫感染。有明显临床表现的寄生虫感染称为寄生虫病。动物机体同时有两种以上虫种寄生，称多寄生现象。

  • 慢性感染是寄生虫病的重要特点之一

生殖内分泌学

  • 激素：是由内分泌腺（无管腺）产生，通过血液循环运送到靶组织、靶器官，调节其生理功能及行为的一类信号分子或者生物活性物质。

  • 激素的分类（按化学结构）：①甾体激素②多肽及蛋白质类激素③胺类激素④脂肪酸类激素（前列腺素)

  • 激素的分类（按产⽣部位及其作⽤）：

    ①松果腺激素:褪黑素MLT、GnRH、TRH、PGs

    ②丘脑下部激素：促性腺激素释放激素GnRH、促甲状腺激素释放激素TRH、GHRH、SST、ACTH

    ③垂体前叶激素:促卵泡素FSH、促黄体素LH、催乳素PRL、GH、HPA、TSH

    ④胎盘激素：（蛋白）马绒毛膜促性腺激素eCG/孕马血清促性腺激素PMSG、人绒毛膜促性腺激素hCG、（甾体）孕酮P4

    ⑤性腺激素：雌激素E2（主要为雌二醇）、孕酮P4、睾酮T、松弛素、抑制素、活化素、卵泡抑素

  • 性腺激素及其生理作用:

    1. E2：①刺激并维持雌性⽣殖道的发育 ②刺激性中枢，使⺟畜发⽣性欲和性兴奋 ③通过正负反馈调节作⽤，调节FSH和LH的分泌 ④刺激垂体前叶分泌促乳素 ⑤促进并维持第⼆性征 ⑥与孕酮协同，刺激和维持乳腺发育⑦怀孕末期雌激素含量升⾼，且与孕酮达到适当比例时，可使催产素对⼦宫肌层发⽣作⽤，为开始分娩创造必需条件

母畜生殖功能的发生发展和发情周期

名词解释

  • 初情期：指母畜开始出现发情现象并发生排卵的时期

  • 性成熟：母畜生长到一定年龄，生殖器官发育完全，生殖机能达到了比较成熟的阶段，基本具备了正常的繁殖功能，达到性成熟

  • 适配年龄：母畜达到体成熟后，身体已发育完全，并且具有雌性成年动物固有外貌特征。在经济实践中，考虑到身体的发育成熟和经济价值，一般选择在性成熟之后体成熟之前，这个适于开始繁殖的年龄称为适配年龄（或繁殖年龄）

  • 发情期：为母畜表现明显的性欲并接受交配的时期

  • 发情周期：雌性哺乳动物达到初情期后，其生殖器官及性行为所呈现的一系列周期性变化称发情周期

  • 排卵：卵泡发育成熟时，卵泡破裂并排出卵子的过程叫排卵，包括卵泡的破裂，卵丘的断裂两个方面。

  • 黄体：卵泡液流出后，卵泡壁塌陷，颗粒层向卵泡腔内形成皱襞内膜结缔组织和血管随之长入颗粒层，使颗粒层脉管化，形成黄体

  • 优势卵泡:从卵泡波中率先发育的卵泡

  • 简答/论述

发情周期分期有几种?简述四期分法中各期的特点

1>四期分法：

  1. 发情前期：也称前情期。
  2. 发情期：为母畜表现明显的性欲并愿意接受交配的时期
  3. 发情后期：卵泡排卵破裂，形成黄体，雌激素开始下降，孕酮尚未分泌
  4. 发情间期：也称间情期。在黄体分泌的孕酮的控制之下。

2>三期分法：

  1. 兴奋期:相当于四期分法的发情期。卵泡发育增大，生殖道发生明显的变化，母畜表现性行为及性兴奋。此期最后为卵泡破裂排卵，发情结束。
  2. 抑制期:相当于四期分法的发情后期和间情期。卵泡破裂后，形成黄体，产生孕酮，外表不表现任何性欲、性兴奋。
  3. 均衡期：相当于四期分法中的发情前期。卵巢中周期黄体开始萎缩，同时新的卵泡开始发育增大。因此，既有退行性变化，又有进行性变化。

3>二期分法：卵泡期（发情前期+发情期）和黄体期（发情后期+间情期）

动物的排卵方式分几类
  1. 自发性排卵。大多数动物为自发性排卵，在每个发情周期中，卵泡发育成熟后不受外界特殊条件刺激的前提下自发排出卵子。根据排卵后黄体形成的机理又可以分成两种：①排卵后自然形成功能性黄体，如牛羊马猪等家畜；②自发性排卵后须经交配才能形成功能性黄体，如啮齿类动物。
  2. 诱导排卵。这类动物卵泡的破裂及排卵须经一定的刺激后才能发生，排卵之前出现的促性腺激素排卵峰也要延迟到适当的刺激之后才能出现。按诱导刺激的性质不同又可以分为两种：①为交配引起排卵的动物，研究最多的有猫和兔；②为精液诱导排卵动物，为驼科动物。
简述排卵过程及排卵机理

排卵过程：

  • 卵泡成熟后，卵泡液增加，卵泡外膜的胶原纤维分解，卵泡壁变薄
  • 卵泡壁中心形成透明的无血管区
  • 卵泡外膜分离，内膜从裂口突出，形成乳头状的排卵点
  • 排卵点膨胀，顶部发生局部贫血
  • 卵泡上皮细胞死亡，并释放出水解酶
  • 卵泡壁破裂
  • 卵子排出

排卵机理：

  1. 卵泡成熟时，GnRH水平升高，促使FSH、LH分泌，出现LH分泌峰（与FSH达到一定比例）后，经过一定时间（羊26小时、牛3－4小时、猪40小时、狗24小时）发生排卵。另外，导致卵泡（包括卵母细胞）结构和功能发生改变。
  2. 卵泡成熟时，PGF2α水平亦升高，使卵泡的血流量增加，卵泡液增多，产生大量的透明质酸，对卵泡上皮白膜和卵泡上皮的破裂起主要作用。
  3. 卵泡成熟时，颗粒细胞内产生纤维蛋白溶解酶原激活剂和cAMP，前者使纤维蛋白溶解酶原变成纤维蛋白溶解酶，后者是促性腺激素起作用的重要细胞内信号物质。
  4. 卵巢平滑肌收缩。
试述黄体的形成和溶解机理
  1. 形成：卵泡液流出后，卵泡壁塌陷，颗粒层向卵泡腔内形成皱襞内膜结缔组织和血管随之长入颗粒层，使颗粒层脉管化。同时，在LH的作用下，颗粒细胞逐渐肥大，形成粒性黄体细胞，构成黄体的主体部分。棱形内膜细胞也增大，变为膜性黄体细胞，位于黄体周围。鞘外膜的结缔组织伸入黄体内形成许多间隔。
  2. 溶解：子宫内膜产生PGF2α，通过血液循环到达卵巢，引起黄体萎缩、溶解。其过程为：黄体细胞发生脂肪变性及空泡化，并逐渐被吸收，毛细血管也萎缩。至下次发情时，黄体迅速缩小，经过2到数周后，体积更小，并被结缔组织所取代，颜色变白，成为白体。最后，白体也被吸收。
以奶牛为例,说明如何才能做到适时配种

受精

受精：是指单倍体的配子（即精子和卵子）相互结合和融合，形成二倍体合子（或称为受精卵）的过程。

授精：人为将精子置于体内生殖道中或体外培养液中，以达到受精目的的操作。

精子获能：哺乳动物刚射出的精子尚不具备受精的能力，只有在雌性生殖道内运行过程中发生进一步充分成熟的变化后，才获得受精能力，这种现象称为精子获能

顶体反应：指精子头部前端与卵子透明带接触后，通过配体-受体相互作用，顶体质膜和顶体外膜融合，顶体内小泡囊泡化，顶体内膜暴露，顶体内酶被激活并释放的胞吐过程。

皮质反应：精子人卵后，激发卵子质膜下皮质颗粒发生胞吐，其内容物从精子人卵点开始迅速向四周扩散，导致透明带、卵子质膜和卵周隙发生变化，阻止多精子受精，这称为皮质反应。

透明带反应：精卵融合，发生皮质反应时，皮质颗粒内容物中酶类引起透明带中糖蛋白发生生化和结构变化，阻止多精子入卵，这称为透明带反应

卵子质膜反应：精子质膜与卵子质膜融合使卵子质膜发生变化，阻止多精子受精，这一过程称为卵子质膜反应。

受精过程
  1. 精子穿过卵丘细胞
  2. 精子穿过透明带
  3. 皮质反应
  4. 透明带反应
  5. 卵质膜反应
  6. 卵子激活
  7. 原核发育与融合
