生殖内分泌学

  • 激素：是由内分泌腺（无管腺）产生，通过血液循环运送到靶组织、靶器官，调节其生理功能及行为的一类信号分子或者生物活性物质。

  • 激素的分类（按化学结构）：①甾体激素②多肽及蛋白质类激素③胺类激素④脂肪酸类激素（前列腺素)

  • 激素的分类（按产⽣部位及其作⽤）：

    ①松果腺激素:褪黑素MLT、GnRH、TRH、PGs

    ②丘脑下部激素：促性腺激素释放激素GnRH、促甲状腺激素释放激素TRH、GHRH、SST、ACTH

    ③垂体前叶激素:促卵泡素FSH、促黄体素LH、催乳素PRL、GH、HPA、TSH

    ④胎盘激素：（蛋白）马绒毛膜促性腺激素eCG/孕马血清促性腺激素PMSG、人绒毛膜促性腺激素hCG、（甾体）孕酮P4

    ⑤性腺激素：雌激素E2（主要为雌二醇）、孕酮P4、睾酮T、松弛素、抑制素、活化素、卵泡抑素

    ⑥神经垂体（垂体后叶）激素:催产素OXT/OT、加压素（抗利尿素）AVP

    ⑦局部激素：前列腺素PGs（类十二烷酸：前列腺素、血栓素、白细胞三烯）

    ⑧外激素

  • 性腺激素及其生理作用:

    1. E2：①刺激并维持雌性⽣殖道的发育 ②刺激性中枢，使⺟畜发⽣性欲和性兴奋 ③通过正负反馈调节作⽤，调节FSH和LH的分泌 ④刺激垂体前叶分泌促乳素 ⑤促进并维持第⼆性征 ⑥与孕酮协同，刺激和维持乳腺发育⑦怀孕末期雌激素含量升⾼，且与孕酮达到适当比例时，可使催产素对⼦宫肌层发⽣作⽤，为开始分娩创造必需条件
    2. P4：①维持妊娠 ②调节发情行为；大量孕酮拮抗E2抑制发情，少量则与E2协同，增强发情表现 ③促进乳腺发育 ④促进生殖道的发育
    3. T：①刺激并维持公畜的性⾏为 ②在FSH和LH的共同作⽤之下，维持精⼦的⽣成 ③刺激和维持附睾的发育，并维持精⼦在附睾中的存活时间 ④刺激并维持副性腺及⽣殖器官的⽣⻓及其功能 ⑤使公畜表现第⼆性征 ⑥对丘脑下部和垂体前叶发挥负反馈作⽤
    4. 四种性腺多肽

  • FSH、LH、eCG、hCG的产生部位、生理作用、临床应用：

    产生部位 生理作用 临床应用
    FSH 垂体前叶 ①刺激卵泡生长发育 ②与LH协同诱导排卵 ③与LH协同产生雌激素 ④刺激卵巢生长，增加卵巢重量 ⑤刺激曲细精管上⽪和次级精⺟细胞发育 ⑥与LH和雄激素协同作⽤, 使精子发育成熟 ⑦能促使足细胞释放精子细胞 用于超排及治疗家畜不育

    外激素：动物向周围环境释放的化学物质，作为信息，引起同类动物⾏为和（或）⽣理上的特定反应。

母畜生殖功能的发生发展和发情周期

名词解释

  • 初情期：指母畜开始出现发情现象并发生排卵的时期

  • 性成熟：母畜生长到一定年龄，生殖器官发育完全，生殖机能达到了比较成熟的阶段，基本具备了正常的繁殖功能，达到性成熟

  • 适配年龄：母畜达到体成熟后，身体已发育完全，并且具有雌性成年动物固有外貌特征。在经济实践中，考虑到身体的发育成熟和经济价值，一般选择在性成熟之后体成熟之前，这个适于开始繁殖的年龄称为适配年龄（或繁殖年龄）

  • 发情期：为母畜表现明显的性欲并接受交配的时期

  • 发情周期：雌性哺乳动物达到初情期后，其生殖器官及性行为所呈现的一系列周期性变化称发情周期

  • 排卵：卵泡发育成熟时，卵泡破裂并排出卵子的过程叫排卵，包括卵泡的破裂，卵丘的断裂两个方面。

  • 黄体：卵泡液流出后，卵泡壁塌陷，颗粒层向卵泡腔内形成皱襞内膜结缔组织和血管随之长入颗粒层，使颗粒层脉管化，形成黄体

  • 优势卵泡:从卵泡波中率先发育的卵泡

  • 简答/论述

发情周期分期有几种?简述四期分法中各期的特点

1>四期分法：

  1. 发情前期：也称前情期。
  2. 发情期：为母畜表现明显的性欲并愿意接受交配的时期
  3. 发情后期：卵泡排卵破裂，形成黄体，雌激素开始下降，孕酮尚未分泌
  4. 发情间期：也称间情期。在黄体分泌的孕酮的控制之下。

2>三期分法：

  1. 兴奋期:相当于四期分法的发情期。卵泡发育增大，生殖道发生明显的变化，母畜表现性行为及性兴奋。此期最后为卵泡破裂排卵，发情结束。
  2. 抑制期:相当于四期分法的发情后期和间情期。卵泡破裂后，形成黄体，产生孕酮，外表不表现任何性欲、性兴奋。
  3. 均衡期：相当于四期分法中的发情前期。卵巢中周期黄体开始萎缩，同时新的卵泡开始发育增大。因此，既有退行性变化，又有进行性变化。

3>二期分法：卵泡期（发情前期+发情期）和黄体期（发情后期+间情期）

动物的排卵方式分几类
  1. 自发性排卵。大多数动物为自发性排卵，在每个发情周期中，卵泡发育成熟后不受外界特殊条件刺激的前提下自发排出卵子。根据排卵后黄体形成的机理又可以分成两种：①排卵后自然形成功能性黄体，如牛羊马猪等家畜；②自发性排卵后须经交配才能形成功能性黄体，如啮齿类动物。
  2. 诱导排卵。这类动物卵泡的破裂及排卵须经一定的刺激后才能发生，排卵之前出现的促性腺激素排卵峰也要延迟到适当的刺激之后才能出现。按诱导刺激的性质不同又可以分为两种：①为交配引起排卵的动物，研究最多的有猫和兔；②为精液诱导排卵动物，为驼科动物。
简述排卵过程及排卵机理

排卵过程：

  • 卵泡成熟后，卵泡液增加，卵泡外膜的胶原纤维分解，卵泡壁变薄
  • 卵泡壁中心形成透明的无血管区
  • 卵泡外膜分离，内膜从裂口突出，形成乳头状的排卵点
  • 排卵点膨胀，顶部发生局部贫血
  • 卵泡上皮细胞死亡，并释放出水解酶
  • 卵泡壁破裂
  • 卵子排出

排卵机理：

  1. 卵泡成熟时，GnRH水平升高，促使FSH、LH分泌，出现LH分泌峰（与FSH达到一定比例）后，经过一定时间（羊26小时、牛3－4小时、猪40小时、狗24小时）发生排卵。另外，导致卵泡（包括卵母细胞）结构和功能发生改变。
  2. 卵泡成熟时，PGF2α水平亦升高，使卵泡的血流量增加，卵泡液增多，产生大量的透明质酸，对卵泡上皮白膜和卵泡上皮的破裂起主要作用。
  3. 卵泡成熟时，颗粒细胞内产生纤维蛋白溶解酶原激活剂和cAMP，前者使纤维蛋白溶解酶原变成纤维蛋白溶解酶，后者是促性腺激素起作用的重要细胞内信号物质。
  4. 卵巢平滑肌收缩。
试述黄体的形成和溶解机理
  1. 形成：卵泡液流出后，卵泡壁塌陷，颗粒层向卵泡腔内形成皱襞内膜结缔组织和血管随之长入颗粒层，使颗粒层脉管化。同时，在LH的作用下，颗粒细胞逐渐肥大，形成粒性黄体细胞，构成黄体的主体部分。棱形内膜细胞也增大，变为膜性黄体细胞，位于黄体周围。鞘外膜的结缔组织伸入黄体内形成许多间隔。
  2. 溶解：子宫内膜产生PGF2α，通过血液循环到达卵巢，引起黄体萎缩、溶解。其过程为：黄体细胞发生脂肪变性及空泡化，并逐渐被吸收，毛细血管也萎缩。至下次发情时，黄体迅速缩小，经过2到数周后，体积更小，并被结缔组织所取代，颜色变白，成为白体。最后，白体也被吸收。
以奶牛为例,说明如何才能做到适时配种

受精

受精：是指单倍体的配子（即精子和卵子）相互结合和融合，形成二倍体合子（或称为受精卵）的过程。

授精：人为将精子置于体内生殖道中或体外培养液中，以达到受精目的的操作。

精子获能：哺乳动物刚射出的精子尚不具备受精的能力，只有在雌性生殖道内运行过程中发生进一步充分成熟的变化后，才获得受精能力，这种现象称为精子获能

顶体反应：指精子头部前端与卵子透明带接触后，通过配体-受体相互作用，顶体质膜和顶体外膜融合，顶体内小泡囊泡化，顶体内膜暴露，顶体内酶被激活并释放的胞吐过程。

皮质反应：精子人卵后，激发卵子质膜下皮质颗粒发生胞吐，其内容物从精子人卵点开始迅速向四周扩散，导致透明带、卵子质膜和卵周隙发生变化，阻止多精子受精，这称为皮质反应。

透明带反应：精卵融合，发生皮质反应时，皮质颗粒内容物中酶类引起透明带中糖蛋白发生生化和结构变化，阻止多精子入卵，这称为透明带反应

卵子质膜反应：精子质膜与卵子质膜融合使卵子质膜发生变化，阻止多精子受精，这一过程称为卵子质膜反应。

妊娠

妊娠：是指从受精开始，经由受精卵阶段、胚胎阶段、胎儿阶段，直至分娩（妊娠结束）的整个生理过程。

妊娠期：是指胎生动物胚胎和胎儿在子宫内完成生长发育的时期，通常是指从最后一次有效配种直至分娩为止（妊娠结束）所经历的时间。

动物的妊娠期：牛282、马340、猪114、猫58、狗62

妊娠识别：指孕体（Conceptus）向母体系统发出其存在的信号，阻止黄体退化，使其继续维持并分泌孕激素，从而使妊娠能够确立并维持下去的一种生理机制。

妊娠建立：继妊娠识别后，母子产生信息和物质交换，妊娠得以维持的现象。

分娩

  • 分娩：妊娠期满，胎儿发育成熟，母体将胎儿及其附属物从子宫内排出体外，这个生理过程称为分娩。

  • 产道：是胎儿排出的必经之道，由软产道和硬产道共同构成。软产道由子宫颈、阴道、前庭及阴门构成。硬产道就是骨盆。

  • 诱导分娩：

  • 分娩预兆

  • 分娩启动的因素：

  1. 内分泌因素：①胎儿内分泌变化：胎儿丘脑下部-垂体-肾上腺轴系对分娩发动起着决定性作用。② 母体内分泌变化：P4↓、E↑、F↑、PGs↑、OT↑、RLX↑。
  2. 机械性因素：胎儿剌激。
  3. 神经性因素：胎儿剌激促进母体神经激素分泌.
  4. 免疫性因素：妊娠末期排斥反应增强。

  • 决定分娩的要素：产力、产道、胎儿与产道之间的关系。

  • 产力（expulsive forces）：将胎儿从子宫中排出的力量称为产力，可以分为阵缩和努责两种力。

    阵缩 ：指子宫肌的收缩，由于子宫肌收缩具阵发性，故称“阵缩”，又称阵痛。是分娩过程中的主要动力，占90％左右。一阵阵有节律的收缩，是一种无意识的表现。
    阵缩的原因：雌激素、孕激素、催产素、松弛素、子宫收缩素、PGs等对子宫肌的调节。
    阵缩生理意义：子宫血管所受的压迫时紧时松，以保证胎儿在娩出前不致窒息，保证血流、氧的供应充足。
    努责：腹肌和膈肌的收缩，称努责。一般只出现在胎儿娩出期。努责是一种有意识的行为。

  • 胎儿与产道的关系：指胎儿在子宫内或产道的状况。包括胎向、胎位、胎势。

    （一）胎向 ：即胎儿的方向，胎儿的身体纵轴与母体纵轴的关系。
    ①纵向：胎儿的纵轴与母体纵轴相互平行。
    正生――胎儿方向与母体方向相反――胎头前置
    倒生――胎儿方向与母体方向相同――胎尾前置。
    ②横向：胎儿纵轴与母体纵轴呈水平垂直，即横卧于子宫内。
    背部前置横向――背横向
    腹部前置横向――腹横向
    身体的一侧朝向产道――侧横向
    ③竖向：胎儿的纵轴向上，与母体纵轴呈上下交叉垂直。
    分背竖向、腹竖向
    （二）胎位： 胎儿的位置，即胎儿背部与母体背部或腹部的关系：
    ①上位：背荐位，胎儿伏卧于子宫内，背部在上；
    ②下位：背耻位，胎儿仰卧于子宫内，背部在下；
    ③侧位：背骼位，胎儿侧卧于子宫内，背部朝向一侧。
    （三）胎势：胎儿在分娩时的姿势，是伸直的，还是屈曲的。

    （四）正常分娩时的胎位、胎向、胎势
    ①胎向――纵向、正生，占大多数；倒生。
    ②胎位――上位、侧位：看侧转程度，一般小于90度，可以通过子宫收缩时的矫正及进入产道时的位置改变，转为上位。大于90度则难于矫正。
    ③胎势：正生――两前肢伸直，头置于两前肢上。

               倒生――两后肢伸直，先进入产道。

    且各个关节是伸展的，而不是屈曲。（只要有一个关节屈曲，就视为异常。

妊娠期疾病

妊娠期疾病：较为常见的有：流产、胎水过多、妊娠水肿、。妊娠截瘫、阴道脱出、妊娠毒血症等

流产：由于胎儿或母体异常而导致妊娠的生理过程扰乱，或它们之间的正常关系受到破坏而导致的妊娠中断，因此排出不能独立生活的孕体。

早产、死产：如果母体在怀孕期满之前排出成活但未成熟的胎儿，称为早产。如果在分娩时排出死亡的胎儿，则称为死产。

引起流产的原因：分表中三类，每类又可分为自发性流产和症状性流产

自发性流产：胎儿及胎盘发生反常或直接受到影响而发生的流产

症状性流产：孕畜某些疾病的一种症状，或是饲养管理不当导致的结果

自发性流产 症状性流产
普通流产（非传染性流产） ①胎盘及胎膜异常②胚胎过多 ③胚胎发育停滞 ①生殖器官疾病 ②激素失调 ③非传染性全身疾病 ④饲养性流产 ⑤中毒性流产 ⑥损伤性及管理性流产 ⑦医疗错误性流产
传染性流产 — —
寄生虫性流产 — —

流产的症状：①隐形流产 ②排出不足月的活胎儿 ③排出死亡而未经变化的胎儿 ④延期流产（包括胎儿干尸化、胎儿气肿、胎儿浸溶）

延期流产：又称死胎停滞，胎儿死亡后由于阵缩微弱，子宫颈不开放或开放不大，死后长期停留于子宫内。依据子宫颈是否开放和黄体是否萎缩，可可发生胎儿干尸化（不萎缩、不开放）和胎儿浸溶（萎缩，开放）

胎儿干尸化：胎儿死亡，未被排出，组织中的水分和胎水被吸收，变为棕黑色，呈干尸样。

胎儿浸溶：妊娠中断后，死亡胎儿的软组织分解，变为液体流出，而骨骼则留在子宫内，称为胎儿浸溶。

妊娠截瘫：是妊娠末期孕畜既无导致截瘫的局部因素，又无明显的全身症状，但后肢不能站立的一种疾病。

子宫脱和阴道脱的鉴别诊断

分娩期疾病（难产）

难产：是指各种原因引起的分娩第一阶段（宫颈开放期）、尤其是第二阶段（胎儿排出期）明显延长，如不进行助产，则母体难于或不能排出胎儿的疾病。

难产的直接病因：

  • 母体性难产：
    1. 产力性难产：子宫迟缓、努责过强及破水过早
    2. 产道性难产：子宫捻转、子宫颈开张不全、骨盆狭窄、阴道，阴门及前庭狭窄、双子宫颈、子宫疝
  • 胎儿性难产：胎向胎位胎势异常、胎儿过大、胎儿畸形

助产术及其适应症：药物催产、矫正胎儿、矫正子宫、扩张产道

  • 用于胎儿：
    1. 牵引术：子宫迟缓；矫正术之后；截胎术后拉出身体；胎儿过大；初产母畜产道狭小；产道被脂肪、肿瘤堵塞；胎儿倒生；多胎动物子宫内仅剩1~2个胎儿而可能发生继发性子宫迟缓时等
    2. 矫正术：通过推拉、扭正、翻转、矫正或拉直胎儿四肢的方法，把异常胎向胎位胎势矫正到正常的助产手术
    3. 截胎术：为了缩小胎儿体积而肢解或除去胎儿身体某部分的手术。难产时胎儿已经死亡，无法拉出胎儿，又不能或不宜剖宫产。
  • 用于母体：
    1. 药物催产
    2. 剖宫产术：无法拉出胎儿或无条件进行截胎尤其在胎儿还活着时

子宫捻转的概念及其治疗法：是指整个子宫、一侧子宫角或子宫角的一部分围绕自己的纵轴发生的扭转。

阴唇的肿胀与捻转方向相反

治疗：主要转正子宫，若在临产时发生则矫正后拉出胎儿

  1. 产道或直肠矫正胎儿及子宫：产道矫正主要目的在借助胎儿矫正捻转的子宫，其是否成功取决于产道开张程度和胎儿是否活着。站立保定，前低后高
  2. 翻转母体：迅速向捻转方向翻转母畜身体
  3. 剖腹矫正
  4. 剖宫产

母畜科学

不育：动物受到不同因素的影响，生育力严重受损或被破坏而导致的绝对不能繁殖。

不孕：由于各种因素而使母畜的生殖机能暂时丧失或降低，称为不孕。

不孕症：为引起各种母畜繁殖障碍的疾病的通俗统称。

不育的原因：先天性不育、

  1. 近亲繁殖与种间杂交不育
  2. 两性畸形
  3. 异性孪生母犊不育
  4. 生殖道畸形

卵巢静止/卵巢功能减退：卵巢机能暂时受到扰乱，处于静止状态，不出现周期性活动母畜有发情的外表症状，但不排卵或延迟排卵

卵巢囊肿：卵巢上有卵泡状结构，直径超过2.5cm，存在时间10天以上，同时卵巢上无正常黄体结构的一种病理状态。

卵泡囊肿：卵泡上皮变性，卵泡壁结缔组织增生变厚，卵细胞死亡，卵泡液未被吸收或增多而形成

卵巢特征：卵巢上无黄体，存在直径特大的卵泡，分泌雌激素
行为表现：无规律频繁发情或持续发情（nymphomania 慕雄狂）
